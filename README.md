# aisignallab-site

Pages légales publiques de l'app OAuth **yt-pipeline** (page d'accueil + politique de
confidentialité), servies par GitHub Pages. Requises pour le passage de l'app en
« Production » et pour la vérification Google.

Le code du pipeline vit dans un dépôt **privé** séparé. Ce dépôt ne contient que des
pages statiques, aucun secret, aucun code métier.

Source de vérité : `deploy/oauth-site/` du dépôt privé du pipeline.
